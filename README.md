# Твой

Статический сайт продуктового магазина «Твой», подготовленный для автоматического деплоя через Netlify.

## Сборка

```bash
npm run build
```

Сборка не минифицирует HTML, CSS и JavaScript и не перекодирует изображения. Она проверяет структуру HTML, JavaScript, JSON-LD, CSS, якоря и локальные ссылки, после чего побайтно копирует production-файлы в `dist/`.

Большие исходные изображения остаются в Git, но не попадают в deploy, потому что страница использует их оптимизированные responsive-версии. Поэтому внешний вид и поведение сайта не меняются, а публикация становится существенно легче.

## Netlify

Настройки уже находятся в `netlify.toml`:

- build command: `npm run build`;
- publish directory: `dist`;
- Node.js: 22.

После подключения репозитория `weeloser/tvoy` к Netlify каждый push в production-ветку автоматически запускает проверку, сборку и атомарный deploy.
